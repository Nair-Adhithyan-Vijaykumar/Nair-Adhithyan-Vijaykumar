## 🔁 Workflow

```bash
$ while true; do
    learn --deep          # Understand the WHY, not just the HOW
    build --with-purpose  # Solve real problems, not toy examples
    break --intentionally # Find failure modes before production does
    analyze --root-cause  # Go beyond surface-level fixes
    improve --systematically
    commit && repeat
  done
```

> *Consistency builds capability. Depth compounds.*
